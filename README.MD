```
│

│

```

## كيفية التشغيل في Termux + Acode

### الطريقة 1: خادم محلي بـ Python (موصى به)
```bash
# في Termux
pkg install python
cd /path/to/quran-hifz
python -m http.server 8080
```
ثم افتح المتصفح على: `http://localhost:8080`

### الطريقة 2: فتح مباشر بـ Acode
- افتح `index.html` داخل Acode
- اضغط "Preview in Browser"

## هيكل ملف JSON للسور

كل سورة تحتاج ملف JSON بالشكل التالي:

```json
{
  "id": 1,
  "name": "اسم السورة",
  "totalAyahs": 7,
  "ayahs": [
    {
      "number": 1,
      "words": [
        {
          "display": "النص العثماني المشكل للعرض",
          "match": "النص المجرد من التشكيل للمطابقة الصوتية"
        }
      ]
    }
  ]
}
```

### قواعد حقل `match`:
- بدون أي تشكيل
- أ/إ/آ → ا
- ة → ه
- ى → ي
- بدون همزات ومدود

## تفعيل التعرف على الصوت

يتطلب:
1. **متصفح Chromium/Chrome** (Firefox لا يدعم Web Speech API)
2. **إذن الميكروفون** عند أول تشغيل
3. لا يشترط الإنترنت لأغلب عمليات التعرف الصوتي المحلي

---

## إضافة سور جديدة

1. أنشئ ملف `surah_XXX.json` في مجلد `data/`
2. استخدم نفس هيكل `surah_001.json` كقالب
3. السور موجودة في `SURAHS_INDEX` داخل `app.js` وتُحمَّل تلقائياً
